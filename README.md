# 📊 Análise de Churn – Telecom X

Projeto de análise exploratória de dados com foco em identificação de fatores relacionados ao churn de clientes.

## 📌 Principais Insights

- Contratos mensais apresentam maior taxa de churn.
- Clientes nos primeiros 12 meses possuem maior risco de cancelamento.
- Pagamento via electronic check possui maior taxa de evasão.
- Clientes com mensalidades mais altas apresentam maior propensão ao churn.

## 🛠 Tecnologias utilizadas

- Python
- Pandas
- Jupyter Notebook
